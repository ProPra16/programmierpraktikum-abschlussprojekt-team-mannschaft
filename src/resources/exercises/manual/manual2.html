Test Driven Development Trainer


Project

In der oberen linken Ecke des TDDTs findet man die Leiste "Project".

Diese weißt folgende Funktionen auf:
1. "Select Exercise": Damit kann man sich eine Aufgabe aus dem Aufgabenkatalog aussuchen.
2. "Create Exercise": Damit kann man sich eine eigene Aufgabe erstellen und diese dann natürlich auch über Select Exercise ausführen.
3. "Open Project": Damit kann man ein bereits angefangenes Project öffnen und weiter daran arbeiten.
4. "Save Project": Damit kann man sein Project sichern.
5. "Run": Damit kann man sein Project Compilen.

1. Es erscheint eine Auswahl der verschiedenen Aufgaben, unter denen du dir eine aussuchen kannst.

2. Es erscheint ein Fenster in dem du ein Exercise erstellen kannst.
Achte dabei bitte darauf, dass du den Titel wie eine Java-Klasse schreibst.
Also der erste Buchstabe groß und im restlichen String keine sonderzeichen.

3. Es erscheint eine Auswahl der verschiedenen Projects die du schon ein angefangen und gesichert hast.
Unter denen du dir dann das gewünschte Project aussuchen und anschließend weiter bearbeiten kannst.

4. Das Project wird gesischert und du erhälst dem bezüglich eine Meldung.

5. Das Project wird Compilier.
