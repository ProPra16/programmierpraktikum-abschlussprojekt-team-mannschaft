Compile

Um zu compilieren sollte man zu allererst das Terminal-Symbol anklicken.
Dadurch öffnet sich unten das Terminal und die Description sowie der Graph auf den man wechseln kann.



Anschließend kann man entweder auf das Play-Symbol klicken, oder über "Project" den Befehl "Run" ausführen.
Nun wird das Project compiliert.
