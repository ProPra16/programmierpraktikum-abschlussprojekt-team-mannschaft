Project

In der oberen linken Ecke des TDDTs findet man die Leiste "Project", wie in Abbildung 1 dargestellt.

(Abb.1)


Die Funktion "Project" des TDDTs weißt folgende Funktionen auf:

1. "Select Exercise": Damit kann man sich eine Aufgabe aus dem Aufgabenkatalog aussuchen.

Es erscheint eine Auswahl (Abb.2) der verschiedenen Aufgaben, unter denen du dir eine aussuchen kannst.

(Abb.2)



2. "Create Exercise": Hiermit können eigene Aufgaben erstellt werden. Diese können dann natürlich auch über Select Exercise ausgeführt werden.

Es erscheint ein Fenster in dem du ein Exercise erstellen kannst.
Achte dabei bitte darauf, dass du den Titel wie eine Java-Klasse schreibst.
Also der erste Buchstabe groß und im restlichen String keine Sonderzeichen.

(Abb.3)



3. "Open Project": Damit kann man ein bereits angefangenes "Project" öffnen und weiter daran arbeiten.

Es erscheint eine Auswahl der verschiedenen Projects die du schon ein angefangen und gesichert hast.
Unter denen du dir dann das gewünschte Project aussuchen und anschließend weiter bearbeiten kannst.

(Abb.4)



4. "Save Project": Durch das ausführen dieser Funktion kann dein "Project" gesichert werden.

Das Project wird gesichert und du erhälst dies bezüglich eine Meldung.

(Abb.5)



5. "Run": Hiermit kann man sein "Project" kompilieren.

Das Project wird kompiliert.

(Abb.6)
