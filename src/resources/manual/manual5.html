Phase

Oben in der Leiste findest du die Befehle "Next Phase" und "Last Phase"
sowie eine Anzeige über die aktuelle Phase "GREEN", "RED" oder "REFACTOR".



Man fängt in der "RED" Phase an und schreibt seinen Test.



Anschließend kann man durch den Befehl "Next Phase" in den "GREEN" Zustand wechseln.
Im "GREEN" Zustand arbeitet man an der Java-Datei.
Diese sollte fehlerfrei sein und mit dem Test kompatibel, damit man in die nächste Phase gelangt.
Ansonsten wird eine Fehlermeldung ausgegeben.





Wenn der Klassencode mit dem Testcode übereinstimmt, kann man in die "REFACTOR" Phase wechseln.



In dieser Phase kann man noch einige Verbesserungen an seinem Code durchführen.
Bei einem weiteren wechsel der Phase landet man wieder bei "RED" und ein neuer Kreislauf beginnt.
